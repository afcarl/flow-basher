# flow-basher

POC Semaphor bot to run shell commands.

## Usage

The script expects these environment variables:

- `SEMAPHOR_BASHER_PASSWORD`
- `SEMAPHOR_BASHER_USERNAME`
- `SEMAPHOR_SPARKLEPALACE_INVITE_CODE`

Then, run either:

    `docker-compose build`
    `docker-compose up`

or

    `python basher.py`
